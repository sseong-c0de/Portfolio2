(About me)

안녕하세요. 사람과 화면 사이를 잇는 셰프 같은 퍼블리셔를 꿈꾸는 이재성입니다.

제조업 경험은 저에게 재료를 손질하는 과정과 같았고, 코딩을 배우는 시간은 손질한 재료로 요리를 완성하는 즐거움이었습니다.

단순히 동작하는 코드가 아닌, 보기 좋고 손질하기 쉬운 요리와 같은 코드를 만들기 위해 끊임없이 배우고 있습니다.

(Skill)

HTML - 국산

웹 접근성과 표준을 고려한 시멘틱한 마크업을 작성할 수 있습니다.

CSS - 국산

반응형 웹과 사용자 중심의 레이아웃 및 스타일링 구현이 가능합니다.

JavaScript - 국내산

동적인 기능 구현과 웹페이지가 자연스럽게 반응하도록 만들 수 있습니다.

jQuery - 국내산

여러 이벤트 처리 및 필요한 기능을 구현할 수 있습니다.

React - 미국산

React를 활용해 코드 재사용과 유지보수에 유리한 코드를 작성할 수 있습니다.

Git - 호주산

협업과 버전 관리를 위한 Git 명령어를 익히고 프로젝트에 적용해보았습니다.

Figma - 호주산

화면 설계부터 실제 인터페이스 흐름을 고려하여 디자인할 수 있습니다.
